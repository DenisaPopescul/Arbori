Parcurgerea arborilor

Parcurgerea unui arbore binar presupune vizitarea fiecărui nod o singură dată, urmând o anumită ordine.


Parcurgerea în inordine

Parcurgerea în inordine (inorder traversal) constă în parcurgerea subarborelui stâng, vizitarea rădăcinii și parcurgerea subarborelui drept.

      
      void parcurgere_inordine(Nod *radacina) {

      if (radacina != nullptr) {

      parcurgere_inordine(radacina->st);

      std::cout << radacina->info << " ";

      parcurgere_inordine(radacina->dr);

      }

      }

Parcurgerea în preordine

Parcurgerea în preordine (preorder traversal) constă în vizitarea rădăcinii, parcurgerea subarborelui stâng și parcurgerea subarborelui drept.

      
      void parcurgere_preordine(Nod *radacina) {

      if (radacina != nullptr) {

      std::cout << radacina->info << " ";

      parcurgere_preordine(radacina->st);

      parcurgere_preordine(radacina->dr);

      }

      }

Parcurgerea în postordine

Parcurgerea în postordine (postorder traversal) constă în parcurgerea subarborelui stâng, parcurgerea subarborelui drept și vizitarea rădăcinii.

      
      void parcurgere_postordine(Nod *radacina) {

      if (radacina != nullptr) {

      parcurgere_postordine(radacina->st);

      parcurgere_postordine(radacina->dr);

      std::cout << radacina->info << " ";

      }

      }
